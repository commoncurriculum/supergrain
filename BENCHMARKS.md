# How to Run Benchmarks

To run the benchmarks and compare `@storable/core` with `solid-js/store`, follow these steps:

1.  **Navigate to the root of the `storable` monorepo.**

2.  **Run the benchmark command:**

    ```bash
    pnpm --filter @storable/core bench
    ```

3.  **Review the output.** The benchmarks will run, and the results will be displayed in the console, showing a direct performance comparison between the two libraries for various operations.
