  - title: Surgical Precision
    details: Track individual properties, not entire objects. When one field changes, only its subscribers update. Your app stays fast at any scale.
  - title: Zero Ceremony
    details: No actions. No reducers. No selectors. Just write to your store like a normal object. Supergrain handles the reactivity.
  - title: Predictable Updates
    details: Bidirectional sync that just works. Changes propagate instantly and consistently, whether from UI events or background processes.
  - title: Framework Freedom
    details: First-class React support with hooks that feel native. Plus adapters for Vue, Svelte, and vanilla JS.
  - title: Deep Reactivity
    details: Nested objects and arrays are tracked automatically to any depth. No special syntax or wrapper functions needed.
  - title: Built for Scale
    details: Powered by alien-signals. Battle-tested architecture that handles complex state graphs without breaking a sweat.
